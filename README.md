### TASK 1 - Distributed System
Simple versioning

## Installation
```bash
cd Client && chmod +x install && ./install
```

## Usage
```bash
## Running server
cd Server; ./runner

## Push code to resource
cd Server/master && client.py push

## Pull code from resource
cd Demo && client.py pull
```
